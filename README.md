# logseq-parent-breadcrumb

If you make hierarchical pages like **\[[Taco/Cat]]**, you may want a quick way to navigate to the parent.

---

A breadcrumb to the parent is included in the title row of the page.

![](/assets/preview1.jpg)
![](/assets/preview2.jpg)



Settings can be configured to display a breadcrumb for the full hierarchy.
